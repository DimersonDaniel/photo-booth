# Vue/Eletron

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```

## Electron

### Debug project
```
npm run electron:serve
```

### Build project
```
npm run electron:build
```
